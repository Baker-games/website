# Bake Game - Simple Website

This is a very basic HTML, CSS and Javascript website for the Bake Games company.  
It contains static pages with simple content and no backend or advanced features.

## Purpose

To display important information about the company. And to act as a personal portfolio to it.

## Infos

First of all to access our raspberry pi you need to be in my wifi, or have my ssh key!

Second, the user I made is called sirmadeira.

Third the public address assigned to it is 192.168.0.15

Fourth to connect just run the command ssh sirmadeira@192.168.0.15

Fifth if setting up nginx you first need to install it sudo apt install nginx

Sixth once installed you can test it by just putting the raspberry pi link http://192.168.0.15/ into your browser
